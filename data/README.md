# Data

The raw and processed datasets used in this project belong to the training provider
and are not published.

Expected local structure for reproducing the notebooks:

```
data/
├── raw/          # original CRM and ad-platform exports
└── processed/    # cleaned datasets produced by notebooks 01_*
```

All notebooks in this repository are stored **with executed outputs**, so the full
analysis — tables, charts and conclusions — is viewable directly on GitHub without
access to the data.
